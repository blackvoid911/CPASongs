## 📐 Technical Specifications

### **Standard Screenshots** (Google Play)
- **Resolution:** 1080×1920 pixels
- **Aspect Ratio:** 9:16 (portrait)
- **File Size:** 11KB - 161KB
- **Format:** PNG, 24-bit
- **Content:** Pure UI, no marketing overlays

### **Marketing Mockups** (Professional)
- **Resolution:** 1200×1600 to 1600×1200 pixels  
- **Aspect Ratio:** Various (optimized per use case)
- **File Size:** 581KB - 751KB
- **Format:** PNG, 24-bit with transparency
- **Content:** Device frames + marketing elements

## 🚀 Quick Setup Instructions

### **For Google Play Console:**
1. Navigate to Google Play Console → Store Listing
2. Upload files `screenshot_1_home.png` through `screenshot_5_bible_reading.png`
3. Add captions if desired
4. Save and publish

### **For Website Use:**
1. Choose appropriate mockup based on page layout
2. Optimize for web if needed (compress for faster loading)
3. Add alt text for accessibility
4. Use responsive images for mobile compatibility

### **For Social Media:**
1. Select mockup based on platform requirements
2. Crop if needed to fit platform specifications
3. Add brand overlay or text as required
4. Test visibility at different sizes
